# 🛒 Sales Performance Analysis

An end-to-end **Sales Data Analysis** project using Python, SQL, and Excel to uncover business insights from retail sales data.

---

## 📌 Project Overview

This project analyses the **Global Superstore Sales Dataset** to identify sales trends, revenue patterns, top-performing products, and customer behaviour — helping businesses make data-driven decisions.

---

## 🎯 Key Business Questions Answered

- 📈 Which region generates the highest revenue?
- 🏆 What are the top 10 best-selling products?
- 📅 Which months have the highest and lowest sales?
- 👥 Which customer segment is most profitable?
- 🚚 How does shipping mode affect profitability?
- 📉 Which product categories have negative profit margins?

---

## 🏆 Key Insights

| Insight | Finding |
|---------|---------|
| Top Region | West region contributes 32% of total revenue |
| Best Category | Technology has highest profit margin (17%) |
| Peak Month | November & December highest sales (holiday season) |
| Best Segment | Consumer segment drives 51% of total sales |
| Worst Category | Furniture has lowest profit margin (2.5%) |

## 🔧 Tech Stack

- **Language:** Python 3.7+
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn, Plotly
- **Database:** SQLite3 (SQL queries for analysis)
- **Spreadsheet:** Excel (data cleaning & pivot tables)
- **IDE:** Jupyter Notebook / Google Colab

## 🚀 How to Run

### 1. Clone the repository
```bash
git clone https://github.com/HemalathaAN/sales-performance-analysis.git
cd sales-performance-analysis
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Get the Dataset
Download from Kaggle:
- [Superstore Sales Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
- Save as `superstore.csv` in the project root

### 4. Run the notebook
```bash
jupyter notebook sales_analysis.ipynb
```

## 📊 Analysis Performed

1. **Data Cleaning & Preprocessing**
   - Handling missing values
   - Date formatting and feature extraction
   - Duplicate removal

2. **Exploratory Data Analysis (EDA)**
   - Sales trends over time
   - Regional performance comparison
   - Category and sub-category analysis

3. **SQL Analysis**
   - Revenue by region using SQL queries
   - Top customers by sales
   - Profitability by product category

4. **Business Insights & Recommendations**
   - Identified underperforming regions
   - Recommended focus product categories
   - Seasonal sales patterns
